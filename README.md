# Blood Report Analyzer using AI

An AI-powered web application that analyzes blood reports using OCR and Large Language Models.

## Features

- Upload PDF/Image
- OCR Extraction
- Blood Parameter Detection
- AI Explanation
- Health Summary
- Download Report

## Tech Stack

### Frontend
- Next.js
- React
- Tailwind CSS

### Backend
- FastAPI
- Python

### AI
- GPT-5 / Gemini

### Database
- PostgreSQL
